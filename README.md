# flower-shop-bot

Telegram-бот для управления цветочным магазином. Контроль сотрудников, фотоотчёты, интеграция с [Posiflora](https://posiflora.com/).

## Возможности

- **Открытие смены** — сотрудник присылает фото, владелец получает уведомление
- **Закрытие смены** — фото + сумма в кассе, автоматический расчёт длительности
- **Фотоотчёты** — любое фото от сотрудника пересылается владельцу
- **Продажи за день** — данные из Posiflora по команде `/sales`
- **Сводка за неделю** — по команде `/week`
- **Кто на смене** — список сотрудников по команде `/staff`
- **Авто-отчёт** — ежедневно в заданное время владелец получает итоги дня

## Команды

### Владелец
| Команда | Описание |
|---|---|
| `/sales` | Продажи за сегодня (из Posiflora) |
| `/week` | Сводка за 7 дней |
| `/staff` | Кто сейчас на смене |

### Сотрудники
| Команда | Описание |
|---|---|
| `/open_shift` | Открыть смену (с фото) |
| `/close_shift` | Закрыть смену (фото + касса) |
| 📸 Фото | Отправить фотоотчёт |

### Служебные
| Команда | Описание |
|---|---|
| `/myid` | Узнать свой Telegram ID |
| `/cancel` | Отменить текущее действие |

## Переменные окружения

| Переменная | Обязательная | Описание |
|---|---|---|
| `TELEGRAM_TOKEN` | ✅ | Токен бота от @BotFather |
| `OWNER_TELEGRAM_ID` | ✅ | Telegram ID владельца (узнать: `/myid`) |
| `POSIFLORA_LOGIN` | ✅ | Email для входа в Posiflora |
| `POSIFLORA_PASSWORD` | ✅ | Пароль Posiflora |
| `POSIFLORA_API_URL` | ✅ | URL API твоего магазина, напр. `https://myshop.posiflora.com/api` |
| `TIMEZONE` | ❌ | Часовой пояс (по умолч. `Europe/Moscow`) |
| `REPORT_HOUR` | ❌ | Час авто-отчёта (по умолч. `21`) |
| `REPORT_MINUTE` | ❌ | Минута авто-отчёта (по умолч. `0`) |

## Как узнать OWNER_TELEGRAM_ID

Напиши боту команду `/myid` — он ответит твоим Telegram ID.

## Как узнать POSIFLORA_API_URL

Это URL твоего магазина в Posiflora. Как правило имеет вид:
```
https://<название-магазина>.posiflora.com/api
```
Уточни в поддержке Posiflora или в настройках аккаунта.

## Деплой на Railway

1. Подключи репозиторий в [Railway](https://railway.com/)
2. В Variables добавь все обязательные переменные
3. Railway автоматически запустит бота
